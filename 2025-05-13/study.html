1. HTML 기본문법

✅ 1.1 요소 Element

- html 요소 element 는 opening tag와 closing tag 그리고 content로 구성

🔗 1.2 중첩 nested element

- 중첩 관계를 다른 말로 하면 부모자식 관계

📁 1.3 주석 Comments

- 주석은 화면 상에 나타나지 않는다.

🌿 1.4 속성 Attribute, 값 Value

- 속성이란 요소의 성질, 특징을 정의한다.

- 속성은 요소에 추가적인 정보, 예를 들어 이미지 파일의 경로, 크기 등과 같은 정보를 제공한다.

- 값 value은 속성에 할당하는 성질이다.
